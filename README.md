# Sakila Assignment

**Author:** Neeka Javeed  
**Date:** 2025-11-08  

## Description
This assignment uses the Sakila database to perform SQL-like queries in R using the `data.table` package. It includes tasks such as data filtering, aggregation, joining, and visualization.  

## Folders
- `data/` : Contains all CSV files from the Sakila database.  
- `scripts/` : Contains R scripts for all tasks.  

## Tasks
1. Display PG films with rental duration > 5  
2. Average rental rate by rating  
3. Count films by language  
4. List customers with their store  
5. Display payments with staff details  
6. Find films not rented  
7. Plot top languages by film count
